1. Onboarding trading

Choisis tes marchés, tes secteurs, ton seuil et la plateforme cible. Les plateformes réelles restent verrouillées tant que les clés API ne sont pas configurées côté serveur.

2. Capital sécurisé

SentiQ ne demande jamais tes identifiants bancaires. Un vrai dépôt devra passer par un prestataire régulé (Stripe, GoCardless, Tink/Plaid selon pays). Ici le dépôt est simulé.

3. Connecteurs réels

Les clés API restent uniquement côté serveur dans `.env.production`. L'interface affiche le statut et permet d'activer un broker quand les secrets sont présents.
